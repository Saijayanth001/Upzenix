# Grade Prediction Flask App

This project predicts student grades using a Random Forest regression model
and a Flask web application.

## Inputs
- Hours Studied
- Attendance
- Previous Score

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Train model
   python train_model.py
3. Run app
   python app.py
4. Open browser at http://127.0.0.1:5000/
